# Keyzy - Adam - 2023

## Prerequisites
* Npm - this is included in the latest version of Node.js

## Getting Started

First install the dependencies:

```bash
npm install
```

Then to run the development server:

```bash
npm run dev
```

Then open `http://localhost:3000` in your browser.

## Tests

To run the test suite:

```bash
npm test
```
